# Python智能停车场车牌识别计费系统 python769

## 项目概述

本项目是一个基于Python语言的智能停车场车牌识别计费系统，集成了车牌识别、计费管理等功能，适用于各类停车场管理场景。系统包含完整的源码、运行说明以及可执行文件。

## 技术栈

本项目主要采用以下技术：

- 编程语言：Python
- 数据处理：Pandas
- 图像识别：OpenCV、TensorFlow
- 数据库：SQLite

## 功能模块

系统主要包括以下功能模块：

- 车牌识别
- 计费管理
- 停车场数据统计
- 用户界面

- 系统管理员：负责系统维护、数据管理等
- 停车场工作人员：负责现场管理与计费
